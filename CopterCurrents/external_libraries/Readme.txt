Download the following external libraries to extend CopterCurrents capabilities:

1) Camera Calibration Toolbox:

	Caltech Matlab library used for Camera calibration, and camera 
	data projection, also implementent in OpenCV.
	
	Author: Jean-Yves Bouguet,jean-yves.bouguet@intel.com. 

	http://www.vision.caltech.edu/bouguetj/calib_doc/
	http://www.vision.caltech.edu/bouguetj/calib_doc/download/index.html

2) deg2utm: 

	Matlab function to convert lat/lon vectors into UTM coordinates (WGS84)

	Author: Rafael Palacios, Universidad Pontificia Comillas, Madrid, Spain

	url => https://www.mathworks.com/matlabcentral/fileexchange/10915-deg2utm
